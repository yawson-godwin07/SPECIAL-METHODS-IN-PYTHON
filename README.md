#  Assignment: Special Methods in Python

## Overview
This repository contains a completed Jupyter Notebook focused on implementing and demonstrating Python special methods (also referred to as magic or dunder methods). The primary objective is to illustrate how user-defined object structures can utilize native Python operators.

---

## Methods Implemented
The uploaded notebook contains distinct practical scripts and commentary blocks mapping out the mechanics of the following 15 special methods:

1. `__init__` – Custom object instantiation and constructor setup
2. `__str__` – Human-readable user string outputs
3. `__repr__` – Official developer system and debugging representations
4. `__len__` – Defining custom object length parameters
5. `__eq__` – Evaluating content equality between instances
6. `__add__` – Overloading the addition (`+`) operator
7. `__sub__` – Overloading the subtraction (`-`) operator
8. `__mul__` – Overloading the multiplication (`*`) operator
9. `__getitem__` – Accessing internal data using bracket indexing syntax
10. `__setitem__` – Modifying internal data via bracket index assignment
11. `__contains__` – Implementing custom membership checks for the `in` operator
12. `__call__` – Allowing an object instance to be executed like a standard function
13. `__iter__` – Initializing a custom iteration protocol loop interface
14. `__bool__` – Determining truth-value evaluation states for logical conditions
15. `__del__` – Destructor lifecycle closure for object cleanups
